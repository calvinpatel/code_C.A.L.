# SUPPLEMENTS.md

Companion to [`CURRICULUM.md`](./CURRICULUM.md). Supplemental learning resources organized by phase, aimed at the **AI Product Engineer → Applied AI/ML Engineer** trajectory.

> **Philosophy:** Project-first. Current tooling. Free-or-cheap by default. Paid only when the mental model is genuinely better. Supplements run at roughly 20–30% of total learning time per phase — the core curriculum and portfolio projects are the main course; these are seasoning.

---

## Phase 1 — Python Fluency

**Already completed:** CS50x · CS50P · Angela Yu 100 Days of Code · Claudia deep-mastery arcs (OOP, Error Handling, File I/O)

### Active additions

- **[Real Python](https://realpython.com)** — Subscription (~$20/mo). Use for 3–4 months then cancel. Reference dojo for engineering practice: virtual environments, packaging, typing/mypy, pytest, decorators, async/await. Dip in as our curriculum surfaces topics.
- **[Obey the Testing Goat](https://www.obeythetestinggoat.com)** — Free online book. Deep TDD companion to engage when our curriculum hits testing.

## Phase 2 — Full-Stack + Web Fundamentals

### Credentialed checkpoint
- **[The Odin Project](https://www.theodinproject.com) — Full-Stack JavaScript path** — Free. The completion artifact is a GitHub portfolio of real projects, which carries more weight with engineers than any paid bootcamp cert.

### Depth boosters
- **[The Joy of React](https://www.joyofreact.com)** — Josh Comeau, ~$300 one-time (wait for sale to ~$200). Teaches the React mental model, not just patterns. Highest-ROI paid resource on this list. The one paid course worth defending.
- **[FastAPI Official Tutorial](https://fastapi.tiangolo.com/tutorial/)** — Tiangolo's docs. Free. Production-grade backend skills. Pair with one real project (TBD in curriculum).

### Breadth expanders
- **[The Missing Semester of Your CS Education](https://missing.csail.mit.edu)** — MIT, free. Shell, git internals, debugging tools, vim basics, regex. ~10 lectures over a few weeks. High ROI for self-taught engineers.
- **SQL — pick one (free):**
  - [SQLBolt](https://sqlbolt.com) — interactive, finishable in a weekend
  - [Mode Analytics SQL Tutorial](https://mode.com/sql-tutorial/) — slightly longer, more analytical depth

## Phase 3 — AI/LLM Integration

The highest-leverage phase for supplements. Field moves fast — fuel from outside the curriculum matters most here.

### Depth boosters
- **[DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/)** — Free, ~1–2 hours each. Built in partnership with Anthropic, OpenAI, LangChain, LlamaIndex. Cherry-pick 5–8 across the phase. Priority picks:
  - *Building Systems with the ChatGPT API*
  - *LangChain for LLM Application Development*
  - *Building and Evaluating Advanced RAG*
  - *Functions, Tools and Agents with LangChain*
  - *Building Agentic RAG with LlamaIndex*
- **[Anthropic Prompt Engineering Interactive Tutorial](https://github.com/anthropics/prompt-eng-interactive-tutorial)** + **[Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook)** — Free. Both education and recon, given Anthropic is on the target list. Learn how Anthropic engineers think about prompting, tool use, and agentic patterns.

### Credentialed checkpoint
- **[Hugging Face NLP Course](https://huggingface.co/learn/nlp-course)** + **[Hugging Face Agents Course](https://huggingface.co/learn/agents-course)** — Free, completion certificate. The real artifact is a trained/finetuned model on HF Hub — that's what matters on the portfolio.

### Breadth expander
- **[Generative AI with Large Language Models](https://www.coursera.org/learn/generative-ai-with-llms)** — DeepLearning.AI / AWS on Coursera. Free to audit. Covers what's *under* the API: pretraining, fine-tuning, RLHF, evaluation. Separates "good at prompting" from "AI Product Engineer."

## Cert Posture

Most certs are low-signal for the target company list (Anthropic, OpenAI, Tempus, Abridge, Epic, Flatiron, PathAI). Hiring is portfolio + interview signal; certs are tiebreakers at best.

- **Defer to Phase 4:** AWS Certified Machine Learning Engineer – Associate. Genuinely useful for applied ML roles, but only meaningful once skills are in place.
- **Skip:** TensorFlow Developer Cert (discontinued 2024), Azure AI-102 (wrong cloud for target list), generic Udemy/Coursera "AI Engineer" certs.
